Exercise 7.
Find the temperature function \(T(x,y)\) in the domain
\begin{equation*} 1\lt r\lt 2,\; 0\lt \theta \lt \frac{\pi}{2} \end{equation*}
that satisfies the following boundary conditions (shown in Figure 10.5.16):
\begin{align*} T(x,y) \amp = 0 \text{ for } x+iy=z=e^{i\theta}, \; 0\lt \theta \lt \frac{\pi}{2};\\ T(x,y) \amp = 50 \text{ for } x+iy=z=2e^{i\theta}, \; 0\lt \theta \lt \frac{\pi}{2};\\ \frac{\partial T}{\partial n} \amp = T_y(x,0) =0 \text{ for } 1 \lt x \lt 2;\\ \frac{\partial T}{\partial n} \amp = T_x(0,y) =0 \text{ for } 1 \lt y \lt 2\text{.} \end{align*}
Solution.
\(T(x,y)=\frac{50}{\ln 2}\ln|z|\text{.}\)
