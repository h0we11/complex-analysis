In this section we investigate some basic ideas concerning sets of points in the plane. The first concept is that of a curve. Intuitively, we think of a curve as a piece of string placed on a flat surface in some type of meandering pattern. More formally, we define a curve to be the range of a continuous complex-valued function \(z(t)\) defined on the interval \([a, b]\text{.}\) That is, a curve \(C\) is the range of a function given by \(z(t) = \big(x(t), y(t)\big) = x(t) +iy(t)\text{,}\) for \(a \le t \le b\text{,}\) where both \(x(t)\) and \(y(t)\) are continuous real-valued functions. If both \(x(t)\) and \(y(t)\) are differentiable, we say that the curve is smooth. A curve for which \(x(t)\) and \(y(t)\) are differentiable except for a finite number of points is called piecewise smooth. We specify a curve \(C\) as
\begin{equation} C:z(t) =x(t) +iy(t) =\big(x(t), y(t)\big) \text{ for } a \le t \le b\text{,}\tag{1.6.1} \end{equation}
and say that \(z(t)\) is a parametrization for the curve \(C\text{.}\) Note that, with this parametrization, we are specifying a direction for the curve \(C\text{,}\) saying that \(C\) is a curve that goes from the initial point \(z(a) =(x(a), y(a)) = x(a)+iy(a)\) to the terminal point \(z(b) = \big(x(b), y(b)\big) = x(b)+iy(b)\text{.}\) If we had another function whose range was the same set of points as \(z(t)\) but whose initial and final points were reversed, we would indicate the curve that this function defines by \(-C\text{.}\)
