Solution 1.3.7.1.

Let \(z=x+iy\text{.}\) Then
\begin{align*} \sqrt{2}|z| \ge |\mathrm{Re}(z)| + |\mathrm{Im}(z)| \amp \text{ iff } \sqrt{2}|z|\ge |x|+|y|\\ \amp \text{ iff } 2|z|^2 \ge |x^2|+2|x|\,|y|+|y^2|\\ \amp \text{ iff } 2x^2+2y^2 \ge |x^2|+2|x|\,|y|+|y^2|\\ \amp \text{ iff } x^2-2|x|\,|y|+y^2 \ge 0\\ \amp \text{ iff } (|x| - |y|)^2 \ge 0\text{,} \end{align*}
which is clearly true. A proper argument will start with this last inequality and work backwards to the appropriate conclusion.
