Solution 2.5.6.1.

Using the result of Example 2.5.3, we need only find the image of the disk \(\overline{D}_1(\frac{1}{2})\) and intersect it with the closed disk \(\overline{D}_1(1)\text{.}\) To begin, we note that
\begin{equation*} \overline{D}_1\left(\frac{1}{2}\right) = \left\{(x, y) : x^2+y^2-x\le \frac{3}{4}\right\}\text{.} \end{equation*}
Because \(z=f^{-1}(w) =\frac{1}{w}\text{,}\) we have, as before,
\begin{align*} u+iv=w\in f\left(\overline{D}_1\left(\frac{1}{2}\right)\right) \amp \Longleftrightarrow f^{-1}(w) \in \overline{D}_1\left(\frac{1}{2}\right)\\ \amp \Longleftrightarrow \frac{1}{w}\in \overline{D}_1\left(\frac{1}{2}\right)\\ \amp \Longleftrightarrow \frac{u}{u^2+v^2}+i\frac{-v}{u^2+v^2}\in \overline{D}_1\left(\frac{1}{2}\right)\\ \amp \Longleftrightarrow \left(\frac{u}{u^2+v^2}\right)^{\!2} + \left(\frac{-v}{u^2+v^2}\right)^{\!2}-\frac{u}{u^2+v^2} \le \frac{3}{4}\\ \amp \Longleftrightarrow \frac{1}{u^2+v^2}-\frac{u}{u^2+v^2}\le \frac{3}{4}\\ \amp \Longleftrightarrow \left(u+\frac{2}{3}\right)^{\!2} + v^2 \ge \left(\frac{4}{3}\right)^{\!2}\text{,} \end{align*}
which is an inequality that determines the set of points in the \(w\) plane that lie on and outside the circle \(C_{\frac{4}{3}}(-\frac{2}{3}) =\{w:|w+\frac{2}{3}|=\frac{4}{3}\}\text{.}\) Note that we do not have to deal with the point at infinity this time, as the last inequality is not satisfied when \((u,v) =(0, 0)\text{.}\) When we intersect this set with \(\overline{D}_1(1)\text{,}\) we get the crescent-shaped region shown in Figure 2.5.7.
