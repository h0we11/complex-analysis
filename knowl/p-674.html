For \(z=re^{i\theta} \ne 0\text{,}\) we can express the function \(w=f(z)=z^2\) in polar coordinates as
\begin{equation*} w=f(z) =z^2=r^2e^{i2\theta}\text{.} \end{equation*}
If we also use polar coordinates for \(w=\rho e^{i\phi}\) in the \(w\) plane, we can express this mapping by the system of equations
\begin{equation*} \rho =r^2 \text{ and } \phi =2\theta \end{equation*}
