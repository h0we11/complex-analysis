Example 5.3.6.

\begin{align*} (i^2)^{i} \amp = \exp \left[ i\log(-1) \right] =e^{-(1+2n) \pi}, \text{ where } n \text{ is an integer, and }\\ (i)^{2i} \amp = \exp(2i\log i) =e^{-(1+4n) \pi}, \text{ where } n \text{ is an integer. } \end{align*}
Since these sets of solutions are not equal, Identity (5.3.8) does not always hold.
