Using Euler’s Formulas (11.1.2) and integration by parts, we obtain
\begin{equation*} a_n=\frac{1}{\pi}\int_{-\pi}^{\pi}\frac{t}{2}\cos nt\,dt = \left.\left(\frac{t\sin nt }{2\pi n} + \frac{\cos nt}{2\pi n^2}\right)\right|_{-\pi}^{\pi} = 0 \text{ for } n=1,2,\ldots\text{,} \end{equation*}
and then using Formula (11.1.3) we get
\begin{align*} b_n \amp = \frac{1}{\pi}\int_{-\pi}^{\pi}\frac{t}{2}\sin nt\,dt = \left.\left(\frac{ -t\cos nt}{2\pi n}+\frac{\sin nt}{2\pi n^2}\right)\right|_{-\pi}^{\pi}\\ \amp =\frac{-\cos n\pi}{n}=\frac{(-1)^{n+1}}{n} \text{ for } n=1,2,\ldots \, \text{.} \end{align*}
