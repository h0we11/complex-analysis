For the Cartesian form, a simple calculation gives
\begin{equation*} f(z) =f(x+iy) =(x+iy)^2 = (x^2-y^2) +i(2xy) =u(x,y) +iv(x,y) \end{equation*}
so that
\begin{equation*} u(x,y) = x^2-y^2, \text{ and } v(x, y) = 2xy \end{equation*}
For the polar form, we refer to Equation (1.5.1) to get
\begin{equation*} f(re^{i\theta})=(re^{i\theta})^2 = r^2e^{i2\theta}=r^2\cos2\theta+ir^2\sin2\theta=U(r,\theta) + iV(r,\theta)\text{,} \end{equation*}
so that
\begin{equation*} U(r,\theta) =r^2\cos2\theta \text{ and } V(r,\theta) =r^2\sin2\theta \end{equation*}
