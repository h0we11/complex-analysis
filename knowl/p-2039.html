We are now ready to define the integral of a complex function along a contour \(C\) in the plane with initial point \(A\) and terminal point \(B\text{.}\) Our approach is to mimic what is done in calculus. We create a partition \(P_n=\{ z_0=A,z_1,z_2,\,.\,.\,.\,,z_n=B\}\) of points that proceed along \(C\) from \(A\) to \(B\) and form the differences \(\Delta z_k=z_k-z_{k-1}\text{,}\) for \(k=1,\,2,\,.\,.\,.\,,\,n\text{.}\) Between each pair of partition points \(z_{k-1}\) and \(z_k\) we select a point \(c_k\) on \(C\text{,}\) as shown in FigureÂ 6.2.6, and evaluate the function \(f\text{.}\) We use these values to make a Riemann sum for the partition:
\begin{equation} S(P_n)=\sum\limits_{k=1}^nf(c_k)(z_k-z_{k-1})=\sum_{k=1}^nf(c_k)\Delta z_k\text{.}\tag{6.2.3} \end{equation}
