The function \(f\) in Equation (8.4.1) is \(f(z) = \frac{\exp (iz) z}{z^2+4}\text{,}\) which has a simple pole at the point \(2i\) in the upper half-plane. Calculating the residue yields
\begin{equation*} \mathrm{Res}[f,2i] =\lim\limits_{z \to 2i}\frac{\exp(iz)z}{z+2i} = \frac{2ie^{-2}}{4i}=\frac{1}{2e^2}\text{.} \end{equation*}
Using Equation (8.4.3) gives
\begin{equation*} \text{ P.V. } \int_{-\infty}^{\infty}\frac{x\sin x}{x^2+4}\,dx = 2\pi \mathrm{Re}(\mathrm{Res}[f,2i]) = \frac{\pi}{e^2} \end{equation*}
