Recall that \(C_2^+(0)\) is the circle \(\{z:|z=2|\}\) with positive orientation. Using partial fraction decomposition gives
\begin{equation*} \frac{2z}{z^2+2}=\frac{2z}{(z+i\sqrt{2}) (z-i\sqrt{2})} = \frac{1}{z+i\sqrt{2}}+\frac{1}{z-i\sqrt{2}}, \text{ so } \end{equation*}
\begin{equation} \int_{C_2^+(0)}\frac{2z}{z^2+2}\,dz = \int_{C_2^+(0)}\frac{1}{z+i\sqrt{2}}\,dz + \int_{C_2^+(0)}\frac{1}{z-i\sqrt{2}}\,dz\text{.}\tag{6.3.14} \end{equation}
The points \(z=\pm i\sqrt{2}\) lie interior to \(C_2^+(0)\text{,}\) so CorollaryÂ 6.3.16 implies that
\begin{equation*} \int_{C_2^+(0)}\frac{1}{z + i\sqrt{2}}\,dz = 2\pi i \text{ and } \int_{C_2^+(0)}\frac{1}{z - i\sqrt{2}}\,dz = 2\pi i\text{.} \end{equation*}
Substituting these values into Equation (6.3.14) yields
\begin{equation*} \int_{C_2^+(0)}\frac{2z}{z^2+2}\ dz=2\pi i+2\pi i=4\pi i \end{equation*}
