If we set \(M=\max\limits_{z \in C}|f(z)|\text{,}\) Equations (7.2.4) and (7.2.5) allow us to conclude that, for all \(z \in C\text{,}\)
\begin{align} \left|\frac{(z_0-\alpha)^{n+1}f(z)}{(z-z_0)(z-\alpha)^{n+1}}\right| \amp = \left|\frac{(z_0-\alpha)^{n+1}}{(z-\alpha)^{n+1}}\right| \left|\frac{f(z)}{(z-z_0)}\right|\notag\\ \amp \le \left(\frac{r}{\rho}\right)^{\!n+1}\left(\frac{1}{\rho -r}\right)M\text{.}\tag{7.2.6} \end{align}
