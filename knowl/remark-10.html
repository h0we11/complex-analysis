Remark 4.3.6.

The equalities given in Example 4.3.5 collectively illustrate an important point with regard to evaluating a geometric series whose beginning index is other than zero. The value of \(\sum\limits_{n=r}^{\infty}z^n\) equals \(\frac{z^{r}}{ 1-z}\text{.}\) If we think of \(z\) as the “ratio” by which any term of the series is multiplied to generate successive terms, we note that the sum of a geometric series equals \(\frac{\;\text{ first term\; } }{\;1-\text{ ratio } \;}\text{,}\) provided \(|\text{ ratio } |\lt 1\text{.}\)
