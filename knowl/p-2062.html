If we use the differentials given in Equation (6.2.4), then we can write Equation (6.2.6) in terms of line integrals of the real-valued functions \(u\) and \(v\text{,}\) giving
\begin{equation} \int_Cf(z)\,dz = \int_Cu\,dx-v\,dy+i\int_Cv\,dx + u\,dy\text{.}\tag{6.2.7} \end{equation}
which is easy to remember if we recall that, symbolically,
\begin{equation*} f(z)\,dz=(u+iv)(dx+i\,dy) \end{equation*}
