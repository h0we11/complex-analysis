Solution 10.6.4.1.

The result of Example 9.4.3 shows that the function \(w=\) \(\mathrm{Arcsin}(z)\) is a conformal mapping of the \(z\) plane slit along the two rays \(x\lt -1, \, y=0\) and \(x>1, \, y=0\) onto the vertical strip \(-\frac{\pi}{2}\lt u\lt \frac{\pi}{2}\text{.}\) Thus, the problem reduces to finding the potential \(\Phi(u,v)\) that satisfies the boundary values
\begin{equation*} \Phi\left(\!-\frac{\pi}{2},v\right) = -300 \text{ and } \Phi\left(\frac{\pi}{2},v\right) = 300 \text{ for all } v\text{.} \end{equation*}
From Example 10.1.1,
\begin{equation*} \Phi(u,v) = \frac{600}{\pi}u \end{equation*}
As in the discussion of Example 10.5.9, the solution in the \(z\) plane is
\begin{align*} \phi(x,y) \amp = \frac{600}{\pi}\mathrm{Re}\big(\mathrm{Arcsin}(z)\big)\\ \amp = \frac{600}{\pi}\mathrm{Arcsin}\left(\frac{\sqrt{(x+1)^2+y^2}-\sqrt{(x-1)^2+y^2}}{2}\right)\text{.} \end{align*}
