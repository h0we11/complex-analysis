Because \(1+i=\sqrt{2}(\cos \frac{\pi}{4}+i\sin \frac{\pi}{4})\text{,}\) we have
\begin{equation*} \arg (1+i) = \left\{\frac{\pi}{4}+2n\pi : n \text{ is an integer } \right\} = \left\{\cdots, -\frac{7\pi}{4}, \frac{\pi}{4}, \frac{9\pi}{4}, \frac{17\pi}{4},\cdots \right\} \end{equation*}
