We use Theorem 4.3.1 and Corollary 4.3.2 to obtain the following representations for the terms on the right side of Equation (7.3.10):
\begin{align} \frac{1}{1+z} \amp = \sum_{n=0}^{\infty}(-1)^nz^n, \amp \amp \text{ valid for } |z| \lt 1,\tag{7.3.11}\\ \frac{1}{1+z} \amp = \sum_{n=1}^{\infty}\frac{(-1)^{n+1}}{z^n}, \amp \amp \text{ valid for } |z| >1,\tag{7.3.12}\\ \frac{1}{2}\left(\frac{1}{1-\frac{z}{2}}\right) \amp = \sum_{n=0}^{\infty}\frac{z^n}{2^{n+1}}, \amp \amp \text{ valid for } |z| \lt 2, \text{ and }\tag{7.3.13}\\ \frac{1}{2}\left(\frac{1}{1-\frac{z}{2}}\right) \amp = \sum_{n=1}^{\infty}\frac{-2^{n-1}}{z^n}, \amp \amp \text{ valid for } |z| >2\text{.}\tag{7.3.14} \end{align}
