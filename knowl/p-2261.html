Since \(f\) is continuous at \(z\text{,}\) for any \(\varepsilon >0\) there is a \(\delta>0\) so that
\begin{equation*} |f(\xi )-f(z)| \lt \varepsilon \text{ when } |\xi -z| \lt \delta\text{.} \end{equation*}
If we require that \(\left\vert \Delta z\right\vert \lt \delta\) and combine this last inequality with Equations (6.4.2) and (6.4.3), and TheoremÂ (6.2.14), we get
\begin{align*} \left|\frac{F(z+\Delta z) -F(z)}{\Delta z} -f(z)\right| \amp = \frac{1}{|\Delta z|} \left\vert \int_{\Gamma }f(\xi)\,d\xi - \int_{\Gamma }f(z)\,d\xi \right\vert\\ \amp \le \frac{1}{|\Delta z|}\int_{\Gamma}\big|f(\xi ) -f(z)\big| \big|d\xi\big|\\ \amp \lt \frac{1}{|\Delta z|}\varepsilon |\Delta z|\\ \amp = \varepsilon \text{.} \end{align*}
