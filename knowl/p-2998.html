We can obtain the antiderivative of \(\frac{1}{1+3\cos^2t}\) by using software such as Mathematica or MAPLE. It is \(\int \frac{1}{1+3\cos^2t}\,dt = \frac{-\mathrm{Arctan}(2\cot t)}{2} = g(t)\text{.}\) Since \(\cot 0\) and \(\cot 2\pi\) are not defined, the computations for both \(g(0)\) and \(g(2\pi)\) are indeterminate. The graph \(s=g(t)\) shown in FigureÂ 8.2.4 reveals another problem: The integrand \(\frac{1}{1+3\cos^2t}\) is a continuous function for all \(t\text{,}\) but the function \(g\) has a discontinuity at \(\pi\text{.}\) This condition appears to be a violation of the fundamental theorem of calculus, which asserts that the integral of a continuous function must be differentiable and hence continuous. The problem is that \(g(t)\) is not an antiderivative of \(\frac{1}{1+3\cos^2t}\) for all \(t\) in the interval \([0,2\pi ]\text{.}\) Oddly, it is the antiderivative at all points except \(0\text{,}\) \(\pi\text{,}\) and \(2\pi\text{,}\) which you can verify by computing \(g\,'(t)\) and showing that it equals \(\frac{1}{1+3\cos^2t}\) whenever \(g(t)\) is defined.
