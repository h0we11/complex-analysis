    Section 5.3 Complex Exponents

    In Section 1.5 we indicated that it is possible to make sense out of expressions such as \(\sqrt{1+i}\) or \(i^{i}\) without appealing to a number system beyond the framework of complex numbers. We now show how this is done by taking note of some rudimentary properties of the complex exponential and logarithm, and then using our imagination.
    We begin by generalizing Identity (5.2.6). Equations (5.2.3) and (5.2.5) show that \(\log(z)\) can be expressed as the set \(\log(z) =\{\mathrm{Log}(z) +i2n\pi : n \text{ is an integer } \}\text{.}\) We can easily show (left as an exercise) that, for \(z \ne 0\text{,}\) \(\exp[\log_{\alpha}(z)]=z\text{,}\) where \(\log_{\alpha}\) is any branch of the function \(\log\text{.}\) But this means that, for any \(\zeta \in \log(z)\text{,}\) the identity \(\exp \zeta =z\) holds true. Because \(\exp[\log(z)]\) denotes the set \(\{\exp \zeta : \zeta \in \log(z) \}\text{,}\) we see that \(\exp[\log(z)]=z\text{,}\) for \(z \ne 0\text{.}\)
    Next, note that Identity (5.2.8) gives \(\log (z^n)=n\log(z)\text{,}\) where \(n\) is any natural number, so that \(\exp[\log(z^n)] = \exp[n\log(z)] = z^n\text{,}\) for \(z \ne 0\text{.}\) With these preliminaries out of the way, we can now come up with a definition of a complex number raised to a complex power.

    Definition 5.3.1. Complex exponent.

    Let \(c\) be a complex number. We define \(z^{c}\) as
    \begin{equation} z^c = \exp\big[c\log(z)\big]\text{.}\tag{5.3.1} \end{equation}
    The right side of Equation (5.3.1) is a set. This definition makes sense because, if both \(z\) and \(c\) are real numbers with \(z>0\text{,}\) Equation (5.3.1) gives the familiar (real) definition for \(z^{c}\text{,}\) as the following example illustrates.

    Example 5.3.2.

    Use Equation (5.3.1) to evaluate \(4^{\frac{1}{2}}\text{.}\)
    Solution.
    Calculating \(4^{\frac{1}{2}}=\exp[\frac{1}{2}\log(4)]\) gives
    \begin{equation} \frac{1}{2}\log(4) =\{\ln 2+in\pi :n \text{ is an integer } \}\text{.}\tag{5.3.2} \end{equation}
    Thus \(4^{\frac{1}{2}}\) is the set \(\{\exp(\ln 2+in\pi) :n \text{ is an integer } \}\text{.}\) The distinct values occur when \(n=0, \, 1\text{.}\) Plugging these values into Equation (5.3.2) gives \(\exp(\ln 2)=2\) and \(\exp(\ln 2+i\pi )=\exp(\ln 2) \exp(i\pi )=-2\text{.}\) In other words, \(4^{\frac{1}{2}} = \{-2,\,2\}\text{.}\)
    The expression \(4^{\frac{1}{2}}\) is different from \(\sqrt{4}\text{,}\) as the former represents the set \(\{-2,\,2\}\) and the latter gives only one value, \(\sqrt{4}=2\text{.}\)
    Because \(\log\) is multivalued, the function \(z^{c}\) will, in general, be multivalued. If we want to focus on a single value for \(z^{c}\text{,}\) we can do so via the function defined for \(z \ne 0\) by
    \begin{equation} f(z) =\exp[c\mathrm{Log}(z)]\text{,}\tag{5.3.3} \end{equation}
    which is called the principal branch of the multivalued function \(z^c\text{.}\) Note that the principal branch of \(z^c\) is obtained from Equation (5.3.1) by replacing \(\log(z)\) with the principal branch of the logarithm.

    Example 5.3.3.

    Find the principal values of \(\sqrt{1+i}\) and \(i^{i}\text{.}\)
    Solution.
    From Example 5.2.3
    \begin{align*} \mathrm{Log}(1+i) \amp = \frac{\ln 2}{2}+i\frac{\pi}{4}=\ln 2^{\frac{1}{2}}+i\frac{\pi}{4}, \text{ and }\\ \mathrm{Log}(i) \amp = i\frac{\pi}{2}\text{.} \end{align*}
    Identity (5.3.3) yields the principal values of \(\sqrt{1+i}\) and \(i^{i}\text{:}\)
    \begin{align*} \sqrt{1+i} \amp = (1+i)^{\frac{1}{2}}\\ \amp = \exp \left[\frac{1}{2}\mathrm{Log}(1+i) \right]\\ \amp = \exp \left[\frac{1}{2}(\ln 2^{\frac{1}{2}}+i\frac{\pi}{4})\right]\\ \amp = \exp\left(\ln 2^{\frac{1}{4}}+i\frac{\pi}{8}\right)\\ \amp = 2^{\frac{1}{4}}\left(\cos \frac{\pi}{8}+i\sin \frac{\pi}{8}\right)\\ \amp \approx 1.09684+0.45509i, \text{ and }\\ i^{i} \amp = \exp[i\mathrm{Log}(i)]\\ \amp = \exp\left[i\left(i\frac{\pi}{2}\right)\right]\\ \amp = \exp\left(\!-\frac{\pi}{2}\right)\\ \amp \approx 0.20788\text{.} \end{align*}
    Note that the result of raising a complex number to a complex power may be a real number in a nontrivial way.
    We now consider the possibilities that arise when we apply Equation (5.3.1).
    Case(i): Suppose that \(c=k\text{,}\) where \(k\) is an integer. Then, if \(z=re^{i\theta} \ne 0\text{,}\)
    \begin{equation*} k\log(z) =\{k\ln(r) +ik(\theta +2n\pi :n \text{ is an integer } \}\text{.} \end{equation*}
    Recalling that the complex exponential function has period \(2\pi i\text{,}\) we have
    \begin{align*} z^k \amp = \exp\big[k\log(z)\big]\\ \amp = \exp\big[k\ln(r) +ik(\theta +2n\pi)\big]\\ \amp = \exp\big[\ln(r^k) +ik\theta +i2kn\pi \big]\\ \amp = \exp\big[\ln(r^k)\big] \exp(ik\theta) \exp(i2kn\pi)\\ \amp = r^k\exp(ik\theta ) = r^k(\cos k\theta +i\sin k\theta)\text{,} \end{align*}
    which is the single-valued \(k\)th power of \(z\) that we discussed in Section 1.5.
    Case(ii): If \(c=\frac{1}{k}\text{,}\) where \(k\) is an integer, and \(z=re^{i\theta} \ne 0\text{,}\) then
    \begin{equation*} \frac{1}{k} \log z = \left\{\frac{1}{k}\ln r+\frac{i(\theta +2n\pi)}{k} : n\text{ is an integer } \right\}\text{.} \end{equation*}
    Hence, Equation (5.3.1) becomes
    \begin{align} z^{\frac{1}{k}} \amp = \exp \left[\frac{1}{k}\log(z)\right]\tag{5.3.4}\\ \amp = \exp \left[ \frac{1}{k}\ln(r) +i\frac{\theta +2n\pi}{k} \right]\notag\\ \amp = r^{\frac{1}{k}}\exp\left(i\frac{\theta +2n\pi}{k}\right)\notag\\ \amp = r^{\frac{1}{k}}\left[\cos\left(\frac{\theta +2n\pi}{k}\right) +i\sin\left(\frac{\theta +2n\pi}{k}\right)\right]\text{.}\notag \end{align}
    When we again use the periodicity of the complex exponential function, Equation (5.3.4) gives \(k\) distance values corresponding to
    \begin{equation*} n=0,\,1,\ldots ,\,k-1\text{.} \end{equation*}
    Therefore, as Example 5.3.2 illustrated, the fractional power \(z^{\frac{1}{k}}\) is the multivalued \(k\)th root function.
    Case(iii): If \(j\) and \(k\) are positive integers that have no common factors and \(c=\frac{j}{k}\text{,}\) then Equation (5.3.1) becomes
    \begin{equation*} z^{\frac{j}{k}} = r^{\frac{j}{k}}\exp\left[i\frac{(\theta +2n\pi) j}{k}\right] =r^{\frac{j}{k}}\left[\cos\left(\frac{(\theta +2n\pi) j}{k}\right) + i\sin\left(\frac{(\theta +2n\pi}{k}\right) \right]\text{.} \end{equation*}
    Again, there are \(k\) distance values that correspond with \(n=0,\,1,\ldots , \, k-1\text{.}\)
    Case(iv): If \(c\) is not a rational number, then there are infinitely many values for \(z^{c}\text{,}\) provided \(z \ne 0\text{.}\)

    Example 5.3.4.

    The values of \(2^{\frac{1}{9}+\frac{i}{50}}\)are
    \begin{align*} 2^{\frac{1}{9}+\frac{i}{50}} \amp = \exp \left[\left(\frac{1}{9}+\frac{i}{50}\right)(\ln 2+i2n\pi ) \right]\\ \amp = \exp \left[ \frac{\ln 2}{9}-\frac{n\pi}{25}+i\left(\frac{\ln 2}{50}+ \frac{2n\pi}{9}\right)\right]\\ \amp = 2^{\frac{1}{9}}e^{-\frac{n\pi}{25}}\left[\cos\left(\frac{\ln 2}{50}+ \frac{2n\pi}{9}\right) +i\sin\left(\frac{\ln 2}{50}+\frac{2n\pi}{9}\right)\right]\text{,} \end{align*}
    where \(n\) is an integer. The principal value of \(2^{\frac{1}{9}+\frac{i}{50}}\) is
    \begin{equation*} 2^{\frac{1}{9}+\frac{i}{50}}=2^{\frac{1}{9}}\left[\cos\left(\frac{\ln 2}{50}\right) + i\sin\left(\frac{\ln 2}{50}\right) \right] \approx 1.079956+0.014972i \end{equation*}
    Figure 5.3.5 shows the terms for this multivalued expression corresponding to
    \begin{equation*} n=-9,\,-8,\ldots ,-1,\,0,\,1,\ldots ,\,8,\,9\text{.} \end{equation*}
    They exhibit a spiral pattern that is often present in complex powers.
    Some of the rules for exponents carry over from the real case. In the exercises we ask you to show that if \(c\) and \(d\) are complex numbers and \(z \ne 0\text{,}\) then
    \begin{align} z^{-c} \amp = \frac{1}{z^{c}},\tag{5.3.5}\\ z^{c}z^{d} \amp = z^{c+d},\tag{5.3.6}\\ \frac{z^{c}}{z^{d}} \amp = z^{c-d},\tag{5.3.7}\\ (z^{c})^n \amp = z^{cn}\text{,}\tag{5.3.8} \end{align}
    where \(n\) is an integer.
    The following example shows that Identity (5.3.8) does not hold if \(n\) is replaced with an arbitrary complex value.

    Example 5.3.6.

    \begin{align*} (i^2)^{i} \amp = \exp \left[ i\log(-1) \right] =e^{-(1+2n) \pi}, \text{ where } n \text{ is an integer, and }\\ (i)^{2i} \amp = \exp(2i\log i) =e^{-(1+4n) \pi}, \text{ where } n \text{ is an integer. } \end{align*}
    Since these sets of solutions are not equal, Identity (5.3.8) does not always hold.
    We can compute the derivative of the principal branch of \(z^{c}\text{,}\) which is the function \(f(z) =\exp[c\mathrm{Log}(z)]\text{.}\) By the chain rule,
    \begin{equation} f\,'(z) =\frac{c}{z}\exp\big[c\mathrm{Log}(z)\big]\text{.}\tag{5.3.9} \end{equation}
    If we restrict \(z^{c}\) to the principal branch, \(z^{c}=\exp\big[c\mathrm{Log}(z)\big]\text{,}\) then Equation (5.3.9) can be written in the familiar form that you learned in calculus. That is, for \(z \ne 0\) and \(z\) not a negative real number,
    \begin{equation*} \frac{d}{dz}z^{c}=\frac{c}{z}z^{c}=cz^{c-1} \end{equation*}
    We can use Identity (5.3.1) to define the exponential function with base \(b\text{,}\) where \(b \ne 0\) is a complex number:
    \begin{equation*} b^z=\exp\big[z\log(b)\big] \end{equation*}
    If we specify a branch of the logarithm, then \(b^z\) will be single-valued and we can use the rules of differentiation to show that the resulting branch of \(b^z\) is an analytic function. The derivative of \(b^z\) is then given by the familiar rule
    \begin{equation} \frac{d}{dz}b^z=b^z\log_{\alpha}(b)\text{,}\tag{5.3.10} \end{equation}
    where \(\log_{\alpha}\) is any branch of the logarithm whose branch cut does not include the point \(b\text{.}\)

    3.

    Show that, if \(z \ne 0\text{,}\) then \(z^{0}\) has a unique value.
    Solution.
    Note that \(0\cdot \log (z) = \{0 \cdot \zeta : \zeta \in \log (z)\}\text{.}\) This set amounts to the single element zero. Thus, for \(z \ne 0, \, z^0 = \exp (0 \cdot \log z) = \exp(0) = 1\text{.}\)

    4.

    For \(z=re^{i\theta} \ne 0\text{,}\) show that the principal branch of
    (a)
    \(z^{i}\) is given by the equation
    \begin{equation*} z^{i}=e^{-\theta}\big[(\ln r) +i\sin(\ln r)\big]\text{,} \end{equation*}
    where \(r>0\) and \(-\pi \lt \theta \le \pi\text{.}\)
    (b)
    \(z^{\alpha}\) (\(\alpha\) a real number) is given by the equation
    \begin{equation*} z^{\alpha}=r^{\alpha}\cos \alpha \theta +ir^{\alpha}\sin \alpha \theta\text{,} \end{equation*}
    where \(r>0\) and \(-\pi \lt \theta \le \pi\text{.}\)

    5.

    Let \(z_n=(1+i)^n\) for \(n=1,\,2,\ldots\) Show that the sequence \(\{z_n\}\) is a solution to the difference equation \(z_n=2z_{n-1}-2z_{n-2}\) for \(n \ge 3\text{.}\)
    Solution.
    \(2z_{n-1}-2z_{n-2}=2(1+i)^{n-1}-2(1+i)^{n-2}=2(1+i)^{n-2}[(1+i)-1]\text{.}\) This last expression simplifies to \(2i(1+i)^{n-2}\text{.}\) Now, \(z_n=(1+i)^n\text{.}\) Since \mathrm{Log} is a one-to-one function, the problem is solved by showing \(\mathrm{Log}[( 1+i)^n] = \mathrm{Log}[2i(1+i)^{n-2}]\text{.}\) Use properties of the logarithm to do this.

    7.

    Does \(1\) raised to any power always equal \(1\text{?}\) Why or why not?
    Solution.
    No. \(1^{a+ib}=e^{a2\pi n}\cos (b2\pi n) + ie^{a2\pi n}\sin (b2\pi n)\text{,}\) where \(n\) is an integer.

    8.

    Construct an example that shows that the principal value of \((z_1z_2)^{\frac{1}{3}}\) need not equal the product of the principal values of \(z_1^{\frac{1}{3}}\) and \(z_2^{\frac{1}{3}}\text{.}\)

    9.

    If \(c\) is a complex number, the expression \(i^c\) may be multivalued. Suppose all the values of \(|i^c|\) are identical. What are these values, and what can be said about the number \(c\text{?}\) Justify your assertions.
    Solution.
    The number \(c\) must be real, and \(|i^{c}|=1\text{.}\)
