We first assume that Statement (2.3.2) is true and show that Statement (2.3.3) is true. According to the definition of limit, for each \(\varepsilon >0\text{,}\) there is a corresponding \(\delta >0\) such that
\begin{equation*} f(z) \in D_{\varepsilon}(w_0) \text{ whenever } z \in D_{\delta}^*(z_0) \end{equation*}
; that is,
\begin{equation*} |f(z)-w_0|\lt \varepsilon \text{ whenever } 0\lt |z-z_0|\lt \delta\text{.} \end{equation*}
Because \(f(z) -w_0=u(x, y) -u_0+i(v(x, y) -v_0)\text{,}\) we can use Inequalities (1.3.2) to conclude that
\begin{equation*} |u(x, y) -u_0| \le |f(z)-w_0| \text{ and } |v(x, y)-v_0| \le |f(z)-w_0|\text{.} \end{equation*}
It now follows that \(|u(x, y) -u_0|\lt \varepsilon\) and \(|v(x, y) -v_0|\lt \varepsilon\) whenever \(0\lt |z-z_0|\lt \delta\text{,}\) and so Statement (2.3.3) is true.
Conversely, assume that Statement (2.3.3) is true. Then for each \(\varepsilon >0\text{,}\) there exists \(\delta _1>0\) and \(\delta_2>0\) so that
\begin{align*} |u(x, y) -u_0|\amp \lt \frac{\varepsilon}{2} \text{ whenever } 0\lt |z-z_0|\lt \delta _1, \ \ \text{ and }\\ |v(x, y) -v_0|\amp \lt \frac{\varepsilon}{2} \text{ whenever } 0\lt |z-z_0|\lt \delta _2\text{.} \end{align*}
We choose \(\delta\) to be the minimum of the two values \(\delta _1\) and \(\delta _2\text{.}\) Then we can use the triangle inequality
\begin{equation*} |f(z) -w_0|\le |u(x, y) -u_0|+|v(x, y)-v_0| \end{equation*}
to conclude that
\begin{equation*} |f(z) -w_0|\lt \frac{\varepsilon}{2}+\frac{ \varepsilon}{2}=\varepsilon \text{ whenever } 0\lt |z-z_0|\lt \delta \end{equation*}
; that is,
\begin{equation*} f(z) \in D_{\varepsilon}(w_0) \text{ whenever } z\in D_{\delta}^*(z_0)\text{.} \end{equation*}
Hence the truth of Statement (2.3.3) implies the truth of Statement (2.3.2), and the proof of the theorem is complete.
