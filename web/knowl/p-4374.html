We now turn our attention to the unit impulse function. First, consider the function \(f_a(t)\) obtained by integrating \(\delta_a(t)\text{:}\)
\begin{equation*} f_a(t) = \int_0^t\delta_a(\tau )\,d\tau = \begin{cases}0 \amp \text{ for } t \lt 0,\\ \frac{t}{a} \amp \text{ for } 0 \le t \le a,\\ 1 \amp \text{ for } a \lt t. \end{cases} \end{equation*}
