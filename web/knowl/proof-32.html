Proof

By the ratio test (check Example 4.4.6), the series in Definition 5.1.1 has an infinite radius of convergence, so \(\exp(z)\) is entire by Theorem 4.4.7, part (i).
Using Theorem 4.4.7, part (ii), we get
\begin{equation*} \exp\,'(z) =\sum\limits_{n=1}^{\infty}\frac{n}{n!} z^{n-1}= \sum\limits_{n=1}^{\infty}\frac{1}{(n-1)!}z^{n-1} = \sum\limits_{n=0}^{\infty}\frac{1}{n!}z^n=\exp(z)\text{,} \end{equation*}
which gives us part (i) of Theorem 5.1.2
To prove part (ii), we let \(\zeta\) be an arbitrary complex number and define \(g(z)\) to be
\begin{equation*} g(z)=\exp(z)\exp(\zeta -z)\text{.} \end{equation*}
Using the product rule, chain rule, and part (i), we have
\begin{equation*} g\,'(z) =\exp(z) \exp(\zeta -z) + \exp(z)[-\exp(\zeta -z)] =0 \text{ for all } z\text{.} \end{equation*}
According to Theorem 3.2.13, this result implies that the function \(g\) must be constant. Thus, for all \(z\text{,}\) \(g(z)=g(0)\text{.}\) Since \(\exp(0)=1\) (verify!), we deduce
\begin{equation*} g(z) =g(0) =\exp(0) \exp(\zeta -0)=\exp(\zeta)\text{.} \end{equation*}
Hence, for all \(z\text{,}\)
\begin{equation*} g(z)=\exp(z) \exp(\zeta -z)=\exp(\zeta)\text{.} \end{equation*}
Setting \(z=z_1\) and letting \(\zeta =z_1+z_2\text{,}\) we get
\begin{equation*} \exp(z_1) \exp(z_1+z_2-z_1) =\exp(z_1+z_2)\text{,} \end{equation*}
which simplifies to our desired result.
To prove part (iii), we let \(\theta\) be a real number. By Definition 5.1.1,
\begin{align*} e^{i\theta} \amp = \exp(i\theta )\\ \amp = \sum_{n=0}^{\infty}\frac{1}{n!}(i\theta)^n\\ \amp = \sum_{n=0}^{\infty}\left[\frac{1}{(2n)!} (i\theta)^{2n}+\frac{1}{(2n+1)!}(i\theta)^{2n+1}\right] {\text{ (separating odd and even exponents) } }\\ \amp = \sum_{n=0}^{\infty}\left[\frac{1}{(2n)!}(i^2)^n\theta^{2n} + \frac{1}{(2n+1)!}i(i^2 )^n\theta^{2n+1}\right]\\ \amp = \sum_{n=0}^{\infty}(-1)^n \frac{\theta^{2n}}{(2n)!} + i\sum_{n=0}^{\infty}(-1)^n\frac{\theta^{2n+1}}{(2n+1)!}\\ \amp = \cos \theta +i\sin \theta {\text{ (by the series representations for the real-valued sine and cosine) } }\text{.} \end{align*}
