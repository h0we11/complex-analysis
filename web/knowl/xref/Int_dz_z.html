Example 6.4.8.

We let \(D=\{z=re^{i\theta}:r>0 \text{ and } -\pi \lt \theta \lt \pi \}\) be the simply connected domain shown in Figure 6.4.9.
We know that \(f(z)=\frac{{1}}{z}\) is analytic in \(D\) and has an antiderivative \(F(z)=\mathrm{Log}(z)\text{,}\) for all \(z\in D\text{.}\) If \(C\) is a contour in \(D\) that joins the point \(z_1\) to the point \(z_2\text{,}\) then Theorem 6.4.5 implies that
\begin{equation*} \int_C\frac{1}{z}\,dz = \int_{z_1}^{z_2}\frac{1}{z}\,dz = \mathrm{Log}(z_2)-\mathrm{Log}(z_1) \end{equation*}
