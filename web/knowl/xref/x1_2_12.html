Exercise 12.

Verify that, if \(z=(x, y)\text{,}\) with \(x\) and \(y\) not both 0, then \(z^{-1}= \frac{{(1, 0)}}{z} \; \big(\text{ i.e., } z^{-1}=\frac{{1}}{z}\big)\text{.}\)
Hint.
Let \(z=(x, y)\) and use the (ordered pair) definition for division to compute \(z^{-1}=\frac{(1, 0)}{(x, y)}\text{.}\) Then, with the result you obtained, use the (ordered pair) definition for multiplication to confirm that \(zz^{-1}=(1, 0)=1\text{.}\)
