Exercise 5.
Find the temperature function \(T(x,y)\) in the semi-infinite strip \(-\frac{\pi}{2}\lt x\lt \frac{\pi}{2},\;y>0\) that satisfies the following boundary values (shown in Figure 10.5.14):
\begin{align*} T\left(\frac{\pi}{2},y\right) \amp = 100 \text{ for } y>0;\\ T(x,y) \amp = 0 \text{ for } -\frac{\pi}{2} \lt x \lt \frac{\pi}{2};\\ T\left(-\frac{\pi}{2}, \, y\right) \amp = 100 \text{ for } y > 0\text{.} \end{align*}
Solution.
\(T(x,y)=100+\frac{100}{\pi}\mathrm{Arctan}(\sin z+1)-\frac{100}{\pi}\mathrm{Arctan}(\sin z-1)\text{.}\)
