Example 2.1.3.

Express the function \(f(z) =\overline{z}\mathrm{Re}(z)+z^2 + \mathrm{Im}(z)\) in the form \(f(z) =u(x,y) +iv(x,y)\text{.}\)
Solution.
Using the elementary properties of complex numbers, it follows that
\begin{equation*} f(z) =(x-iy) x+(x^2 - y^2 + i2xy) + y = (2x^2-y^2+y) + i(xy)\text{,} \end{equation*}
so that \(u(x,y) = 2x^2-y^2+y\) and \(v(x,y) = xy\text{.}\)
