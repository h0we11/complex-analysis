Example 11.7.8.

Find \(\mathcal{L}\big(f(t)\big)\) if \(f(t)\) is given in Figure 11.7.9.
Solution.
We can represent \(f(t)\) in terms of step functions:
\begin{equation*} f(t) = 1-U_1(t)+U_2(t)-U_3(t)+U_4(t)-U_5(t)\text{.} \end{equation*}
Using the result of Example 10.6.1 and linearity, we obtain
\begin{equation*} \mathcal{L}\big(f(t)\big) = \frac{1}{s} - \frac{e^{-s}}{s} + \frac{e^{-2s}}{s} - \frac{e^{-3s}}{s} + \frac{e^{-4s}}{s} - \frac{e^{-5s}}{s}\text{.} \end{equation*}
