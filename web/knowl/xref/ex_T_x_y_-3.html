Exercise 4.
Find the temperature function \(T(x,y)\) inside the unit disk \(|z|\lt 1\) that satisfies the following boundary values (shown in Figure 10.5.13).
\begin{align*} T(x,y) \amp = 20 \text{ for } x+iy=z=e^{i\theta}, 0\lt \theta \lt \frac{\pi}{2};\\ T(x,y) \amp = 60 \text{ for } x+iy=z=e^{i\theta}, \frac{\pi}{2} \lt \theta \lt 2\pi \text{.} \end{align*}
\hint{Use \(w=\frac{i(1-z)}{1+z}\text{.}\)}
