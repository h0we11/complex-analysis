Example 2.1.2.

Write \(f(z)=z^4\) in the form \(f(z)=u(x,y) + iv(x,y)\text{.}\)
Solution.
Using the binomial formula, we obtain
\begin{align*} f(z) \amp = (x+iy)^4 = x^4+4x^3iy+6x^2(iy)^2 + 4x(iy)^3 + (iy)^4\\ \amp =(x^4-6x^2y^2+y^4) +i(4x^3y-4xy^3)\text{,} \end{align*}
so that \(u(x,y) =x^4 - 6x^2 y^2 + y^4\) and \(v(x,y) =4x^3 y - 4xy^3\text{.}\)
