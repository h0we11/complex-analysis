Solution 2.3.2.1.

If \(x=r\cos \theta\) and \(y=r\sin \theta\text{,}\) then
\begin{equation*} u(x, y) = \frac{2r^3\cos ^3\theta}{r^2\cos ^2\theta +r^2\sin ^2\theta} = 2r\cos^3\theta\text{.} \end{equation*}
Because \(\sqrt{(x-0)^2+(y-0)^2}=r\) and because \(|\cos ^3\theta |\lt 1\text{,}\)
\begin{equation*} |u(x, y) -0|=2r|\cos ^3\theta |\lt \varepsilon \ \ \text{ whenever } 0\lt \sqrt{x^2+y^2}=r\lt \frac{\varepsilon}{2}\text{.} \end{equation*}
Hence, for any \(\varepsilon >0\text{,}\) Inequality (2.3.1) is satisfied for \(\delta =\frac{\varepsilon}{2}\text{;}\) that is, \(u(x, y)\) has the limit \(u_0=0\) as \((x, y)\) approaches \((0, 0)\text{.}\)
