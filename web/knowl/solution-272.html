Solution 5.2.3.1.

By standard computations, we have
\begin{align*} \log(1+i) \amp = \Big\{\ln |1+i| +i(\mathrm{Arg}(1+i) +2n\pi ) :n\text{ is an integer } \Big\}\\ \amp = \left\{\ln \sqrt{2}+i\left(\frac{\pi}{4}+2n\pi\right) :n\text{ is an integer } \right\}, \text{ and }\\ \log(i) \amp = \Big\{\ln |i| +i\big(\mathrm{ Arg}(i) +2n\pi\big) :n\text{ is an integer } \Big\}\\ \amp = \left\{i\left(\frac{\pi}{2}+2n\pi\right) :n\text{ is an integer } \right\}\text{.} \end{align*}
The principal values are
\begin{align*} \mathrm{Log}(1+i) \amp = {\mathrm{ln}\sqrt{2}+i\frac{\pi}{4}}=\frac{\ln 2}{2}+i\frac{\pi}{4}. \text{ and }\\ \mathrm{Log}(i) \amp = i\frac{\pi}{2}\text{.} \end{align*}
