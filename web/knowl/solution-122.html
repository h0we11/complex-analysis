Solution 2.3.3.1.

If we let \((x, y)\) approach \((0, 0)\) along the \(x\) axis, then
\begin{equation*} \lim\limits_{(x,0) \to (0,0)} u(x, 0) = \lim\limits_{(x,0) \to (0,0)}\frac{(x)(0)}{x^2+0^2} = 0\text{.} \end{equation*}
But if we let \((x, y)\) approach \((0,\,0)\) along the line \(y=x\text{,}\) then
\begin{equation*} \lim\limits_{(x,x) \to (0,0)}u(x, x) = \lim\limits_{(x,x) \to (0, 0)}\frac{(x)(x)}{x^2+x^2}=\frac{1}{2}\text{.} \end{equation*}
Because the value of the limit differs depending on how \((x, y)\) approaches \((0, 0)\text{,}\) we conclude that \(u(x, y)\) does not have a limit asÂ \((x, y)\) approaches \((0, 0)\text{.}\)
