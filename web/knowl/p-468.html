The point \(z_0\) is said to be an interior point of the set \(S\) provided that there exists an \(\varepsilon\)-neighborhood of \(z_0\) that contains only points of \(S\text{;}\) \(z_0\) is called an exterior point of the set \(S\) if there exists an \(\varepsilon\)-neighborhood of \(z_0\) that contains no points of \(S\text{.}\) If \(z_0\) is neither an interior point nor an exterior point of \(S\text{,}\) then it is called a boundary point of \(S\) and has the property that each \(\varepsilon\)-neighborhood of \(z_0\) contains both points in \(S\) and points not in \(S\text{.}\) FigureÂ 1.6.8 illustrates this situation.
