The expression \(\lim\limits_{z \to z_0}f(z) = w_0\) means that for each real number \(\varepsilon > 0\text{,}\) there exists a real number \(\delta >0\) such that
\begin{equation*} |f(z) -w_0|\lt \varepsilon \text{ whenever } 0\lt |z-z_0| \lt \delta\text{.} \end{equation*}
Using the notation of (1.6.4) and (1.6.6), we can express the last relationship as
\begin{equation*} f(z) \in D_{\varepsilon}(w_0) \text{ whenever } z \in D_{\delta}^*(z_0) \end{equation*}
