The curve \(C\) defined by Equation (6.2.1) is said to be a smooth curve if the function \(z\,'\) is continuous and nonzero on the interval. If \(C\) is a smooth curve, then \(C\) has a nonzero tangent vector at each point \(z(t)\text{,}\) which is given by the vector \(z\,'(t)\text{.}\) If \(x\,'(t_0)=0\text{,}\) then the tangent vector \(z\,'(t_0)=iy\,'(t_0)\) is vertical. If \(x\,'(t_0) \ne 0\text{,}\) then the slope \(\frac{dy}{dx}\) of the tangent line to \(C\) at the point \(z(t_0)\) is given by \(\frac{{y\,'(t_0)}}{x\,'(t_0)}\text{.}\) Hence for a smooth curve the angle of inclination \(\theta (t)\) of its tangent vector \(z\,'(t)\) is defined for all values of \(t\in [a,b]\) and is continuous.
