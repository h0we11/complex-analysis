Example 3.3.3.

The function \(f(z)=z^3=x^3-3xy^2+i(3x^2y-y^3)\) is analytic for all values of \(z\text{.}\) Hence it follows that
\begin{equation*} u(x,y) = \mathrm{Re}\left[ f(z) \right] =x^3-3xy^2 \end{equation*}
is harmonic for all \(z\text{,}\) and that
\begin{equation*} v(x,y) = \mathrm{Im}\left[ f(z) \right] =3x^2y-y^3 \end{equation*}
is a harmonic conjugate of \(u(x,y)\text{.}\)
