If the contour \(C\) has two parametrizations
\begin{align*} {2} \amp C : z_1(t) =x_1(t) +iy_1(t), \amp \amp \text{ for } a \le t \le b, \text{ and }\\ \amp C : z_2(\tau) =x_2(\tau) +iy_2(\tau), \amp \amp \text{ for } c \le \tau \le d\text{.} \end{align*}
and there exists a differentiable function \(\tau =\phi (t)\) such that
\begin{equation} c=\phi(a), d=\phi(b), \text{ and } \phi\,'(t)>0, \text{ for } a\lt t\lt b\text{.}\tag{6.2.10} \end{equation}
then we say that \(z_2(\tau )\) is a reparametrization of the contour \(C\text{.}\) If \(f\) is continuous on \(C\text{,}\) then we have
\begin{equation} \int_a^bf\big(z_1(t)\big) z\,_1'(t)\,dt=\int_c^{d}f\big(z_2(\tau)\big) z\,_2'(\tau )\,d\tau\text{.}\tag{6.2.11} \end{equation}
