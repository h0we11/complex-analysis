Suppose that \(f\) and \(g\) are differentiable. From Equation (3.1.1) and the technique exhibited in the solution to ExampleÂ 3.1.1, we can establish the following rules, which are virtually identical to those for real-valued functions.
\begin{align} \frac{d}{dz}C \amp = 0, \text{ where } C \text{ is a constant, and }\tag{3.1.4}\\ \frac{d}{dz}z^n \amp = nz^{n-1}, \text{ where } n \text{ is a positive integer } .\tag{3.1.5}\\ \frac{d}{dz}[Cf(z)] \amp = Cf\,'(z)\tag{3.1.6}\\ \frac{d}{dz}[f(z)+g(z)] \amp = f\,'(z) + g\,'(z)\tag{3.1.7}\\ \frac{d}{dz} [f(z) g(z)] \amp = f(z) g\,'(z) + g(z) f\,'(z)\tag{3.1.8}\\ \frac{d}{dz}\frac{f(z)}{g(z)} \amp = \frac{g(z) f\,'(z) -f(z) g\,'(z)}{[g(z)]^2} \text{ if } g(z) \ne 0, \text{ and }\tag{3.1.9}\\ \frac{d}{dz}\,f\big(g(z)\big) \amp = f\,'\big(g(z)\big) g\,'(z)\text{.}\tag{3.1.10} \end{align}
