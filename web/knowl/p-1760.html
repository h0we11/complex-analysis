Case(ii): If \(c=\frac{1}{k}\text{,}\) where \(k\) is an integer, and \(z=re^{i\theta} \ne 0\text{,}\) then
\begin{equation*} \frac{1}{k} \log z = \left\{\frac{1}{k}\ln r+\frac{i(\theta +2n\pi)}{k} : n\text{ is an integer } \right\}\text{.} \end{equation*}
Hence, Equation (5.3.1) becomes
\begin{align} z^{\frac{1}{k}} \amp = \exp \left[\frac{1}{k}\log(z)\right]\tag{5.3.4}\\ \amp = \exp \left[ \frac{1}{k}\ln(r) +i\frac{\theta +2n\pi}{k} \right]\notag\\ \amp = r^{\frac{1}{k}}\exp\left(i\frac{\theta +2n\pi}{k}\right)\notag\\ \amp = r^{\frac{1}{k}}\left[\cos\left(\frac{\theta +2n\pi}{k}\right) +i\sin\left(\frac{\theta +2n\pi}{k}\right)\right]\text{.}\notag \end{align}
