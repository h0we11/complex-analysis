Solution 6.2.16.1.

We parametrize the semicircle \(C_1\) as
\begin{equation*} C_1:z(t)=-\cos t+i\sin t, \text{ and } dz=(\sin t+i\cos t)\,dt, \text{ for } 0 \le t \le \pi\text{.} \end{equation*}
Applying Theorem 6.2.11, we have \(f(z)=\overline{z}\text{,}\) so
\begin{align*} f\big(z(t)\big) \amp = \overline{z(t)}=\overline{(-\cos t+i\sin t)}=-\cos t-i\sin t, \text{ and }\\ \int_{C_1}\overline{z}\,dz \amp = \int_0^{\pi}(-\cos t-i\sin t) (\sin t+i\cos t)\,dt\\ \amp = -i\int_0^{\pi}(\cos ^2t+\sin ^2t)\,dt = -\pi i\text{.} \end{align*}
We parametrize \(C_2\) in three parts, one for each line segment:
\begin{align*} {4} \amp z_1(t) \amp \amp = -1+it, \amp \amp dz_1 = i\,dt, \amp \amp \text{ and } f\big(z_1(t)\big)=-1-it.\\ \amp z_2(t) \amp \amp = -1+2t+i, \amp \amp dz_2 = 2\,dt, \amp \amp \text{ and } f\big(z_2(t)\big)=-1+2t-i.\\ \amp z_3(t) \amp \amp = 1+i(1-t), \amp \amp dz_3 = -i\,dt, \amp \amp \text{ and } f\big(z_3(t)\big)=1-i(1-t)\text{.} \end{align*}
where \(0 \le t \le 1\) in each case. We get our answer by adding the three integrals along the three segments:
\begin{equation*} \int_{C_2}\overline{z}\ dz=\int_0^1(-1-it) i\,dt+\int_0^1(-1+2t-i) 2\,dt+\int_0^1[1-i(1-t)](-i)\,dt\text{.} \end{equation*}
Separating the right side of this equation into its real and imaginary parts gives
\begin{equation*} \int_{C_2}\overline{z}\,dz=\int_0^1(6t-3)\,dt+i\int_0^1(-4)\,dt=-4i\text{.} \end{equation*}
Note that the value of the contour integral along \(C_1\) isn’t the same as the value of the contour integral along \(C_2\text{,}\) although both integrals have the same initial and terminal points.
