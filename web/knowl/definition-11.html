Definition 1.4.6. \(\mathbf{\mathrm{Arg} \; z}\).

Let \(z \ne 0\) be a complex number. Then
\begin{equation} \mathrm{Arg} \, z= \theta \text{ provided } z=r(\cos \theta +i\sin \theta ) \text{ and } -\pi \lt \theta \le \pi\text{.}\tag{1.4.4} \end{equation}
If \(\theta =\mathrm{Arg}\,z\text{,}\) we call \(\theta\) the argument of \(z\text{.}\)
