The inverse mapping of \(f\text{,}\) which we denote \(g\text{,}\) is then
\begin{equation*} z=g(w) = w^{\frac{1}{2}} = \rho^{\frac{1}{2}}e^{i\frac{\phi}{2}}\text{,} \end{equation*}
where \(w \in B\text{.}\) That is,
\begin{equation*} z = g(w) =w^{\frac{1}{2}} = |w|^{\frac{1}{2}}e^{i\frac{{\mathrm{Arg}}(w)}{2}}\text{,} \end{equation*}
where \(w \ne 0\text{.}\) The function \(g\) is so important that we call special attention to it with a formal definition.
